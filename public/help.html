Site

A Site is a physical location -- an address.

Company

A Company is the name of a company.

Office

An Office is the combination of a Site and Company. This is the mechanism that allows you to configure a company at multiple sites and/or multiple companies at one site.

Typically, you will use the Office concept to designate the location and ownership of assets.

User

User defines the users of the system. They can be:
  • Administrators
  • Support
  • User

Equipment is assigned a User, which then relates it to an office, which relates it to a company and location.

Lookup

The Lookup facility enables you to define choices for many of the drop down lists in the system.

You can define values for:

  • Computer Manufacturers
  • Computer Types
  • Operating Systems
  • Software Publishers
  • Technical Support Ticket Types
  • Backup Tape Destinations

Lookups give your database consisteny and factilitate reporting while giving you the flexibility to customize data in your system.

Computer

Computer defines the physical computer hardware in your system. They are usually servers, desktops, or laptops; but, can be expanded to include switches, routers, or any other piece of hardware you want to track by using the Lookup facility.

For example, lets say I wanted to track staplers. I would first create a new Manufacturer lookup value and then a new Type value. I could then enter my staplers into the database and assign them to users. If I wanted I could also assign asset tags to them and enter that into the database as well.

Computer Allocation

Computer allocations allow you to assign hardware to a specific user. Through the user you know the location of the hardware.

Software

Software defines software products.

Software Allocations

Software Allocations allow you to assign software to specific users.
